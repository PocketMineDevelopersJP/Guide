# 最初に

## 目標

初めに、私達の目標をここでお伝えします。  
私達`server.mcbe.jp`が行いたいのは以下の通りです。  

1. Lobiに代わるコミュニティとして **DiscordとMCBEForumに移行** すること

これらの目標を達成するため、私達は現在以下のプロジェクトを進行中です。
